This is a basic neural network with 4 layers and 8 neurons. Each circle represents a neuron. The number in the middle is the neuron's potential. Lines between each neurons represent a 'connection'. Neurons in layer 1 connect to neurons in layer 2. Neurons in layer 2 connect to neurons in layer 3 and so on.

When you stimulate the first neuron, it raises it's potential. Once a neuron potential reaches 100, it will 'fire', which causes the neurons that it is connected to to be stimulated and raise in potential. There is a short delay before firing to make it easier to display the cascading effect in action. The text box with the green color represents the 'weight' between neurons. How much will one neuron stimulate another neuron when it fires? Try adjusting these weights to see the effect on the network.
